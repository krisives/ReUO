Following is a listing of all server-emu's I know of, sorted by source availability. If you are developer of one of the projects, and have a better type&feature description simply mail me. (And of course I'm also happy to add new one's :o)
UOX UOX is the primary server for Arachnide.
It's Open Source (GPL), has a lot of features, frequently updated&improved (and people and their opinions are respected.)
UOX was one of the first servers around, and in the years it has influenced and inspired all other servers in a major way.

Wolfpack The Wolfpack server is an UOX layoff. The only one that managed to survive over time...
Therefore it's also Open Source released under the GPL license.

SWORDS SWORDS is a server from and with punt. It's not yet finished, however I'm already nosy how it will turn out.
It's released under the QT-License, which is also an approved Open Source license.

Sphere SphereServer is a closed source, non-copyleft server.
Special features are built in web- and email server.

POL Well I'm not sure if POL better fit's in the open section, or here in the closed.
POL is a highly scriptable server. 90% of the server itself is written with scripts, so you can modify a lot of behavior without having a compiler or knowing C. (But you'll have to learn their script language either way). However the core itself is closed source (and thus it's landing here).

FUSE FUSE was a server for windows only. Allowing programmers to 'fuse' in their dll's. However it's dead today.

CFUSE Yust another deserted project?

UWE status unknown...

UOAWE Another dead server...
