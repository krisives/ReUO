Version 0.0.3

  • Graphic engine optimized.
  • Game screen resizeable.
  • Static tiles are diplayed in map.
  • Map integrated into game display.
  • Map is now a own thread.
  • Log file is created.
  • Textra display mode introduced.
  • Player animations.
  • Mode selector.
  • UO-Network (walking&speaking) for textra functional.
  • Remeber to setup your Ultima Online directory right in umelange.ini...
    • Pura java (.jar):

        umelange_003_jar.zip

      Supposed to be executed by a JIT (Just-In-Time) compiler. You will need to download SUN Runtime Environtment 1.3 ( http://java.sun.com ) with the Hotspot-VM, and run this version.

      • You can also use any JIT compiler you like, or native compile it yourself for your platform.
      • If you are able to native compile for any OS please let me now!

    • Source code:

        umelange_003_source.zip

      Well, the project is GNU, so here is the source code. Have a lot of fun.
      (However, please leave the version string to (unofficial).)

    • A note to the microsoft VM:


      The Microsoft VM is no longer supported. It has shown to be unstable and having unpredictable behaviour. Until another (affortable) method of native compiling is found, there are no .exe vesions aviable, sorry for that.
