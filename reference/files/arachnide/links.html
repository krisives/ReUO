Ultima Online Project (UOP)

    There! Also checkout Arachnide's brother project.

    Similarities:
    • Both use SDL as base graphic library.
    • Both are Open Source and released under the Gnu Public License.
    • We are both hosted at SourceForge.
    • UOP also focuses on UOX as it's primary server.
    • We entered public around nearly at the same time. (just 4 days difference :o)

    Differences:
    • UOP is written in C, while Arachnide is object oriented designed in C++.
      (Both C variants have their advantages and disadvantages.)
    • Arachnide focuses on windows, while UOP has linux as development platform.
      However I bet both can and will be easily ported to the 'other' OS.
    • Today (21.11.2000) UOP has already networking implented...


    Simple DirectMedia Layer (SDL)

    Simple DirectMedia Layer is a cross-platform multimedia library designed to provide fast access to the graphics framebuffer and audio device. It is used by MPEG playback software, emulators, and many popular games, including the award winning Linux port of "Civilization: Call To Power." Simple DirectMedia Layer supports Linux, Win32, BeOS, MacOS, Solaris, IRIX, and FreeBSD.
